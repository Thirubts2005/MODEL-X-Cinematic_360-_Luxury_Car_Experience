# MODEL X — Cinematic 360° Luxury Car Experience

A futuristic and cinematic luxury car showcase website built using HTML, CSS, JavaScript, and GSAP animations.

This project delivers a premium automotive web experience with:
- 360° interactive car rotation
- Cinematic intro animation
- Futuristic HUD-inspired UI
- Scroll-based storytelling
- Luxury automotive aesthetic
- High-performance visual effects

---

## 🚀 Features

### 🎥 Cinematic Intro
- Animated loading screen
- Engine initialization sequence
- Dynamic headlight effects
- Smooth GSAP transitions


### 🔄 360° Car Viewer
- Interactive frame-by-frame car rotation
- Scroll-controlled animation
- Drag-to-rotate functionality
- Realistic shadow and reflection effects

### ✨ Futuristic UI
- HUD-style interface
- Animated radial dial system
- Custom glowing cursor
- Cyberpunk-inspired design language

### 📊 Performance Section
- Animated performance metrics
- Dynamic stat bars
- Responsive grid layout

### 📱 Responsive Design
- Mobile optimized
- Adaptive layouts
- Smooth user experience across devices
